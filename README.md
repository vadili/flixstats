FlixStats is a movie analytics dashboard built with **React**, **Tailwind CSS**, and **Recharts**, powered by real-time data from **The Movie Database (TMDB) API**. It provides insights into movie trends, genres, ratings, popularity, and financial performance through interactive data visualizations, all wrapped in a Netflix-inspired dark UI.

---

## Features

- Interactive Charts
  - Genre distribution (Pie Chart)
  - Rating distribution (Bar Chart)
  - Movie release timeline (Line Chart)
  - Top 10 movies by popularity (Bar Chart)
  - Budget vs Revenue ROI analysis (Scatter Plot)

- Key Metrics Dashboard
  - Total box office revenue
  - Average movie rating
  - Total movies analyzed
  - Top-performing genre

- Movie Data Table
  - Ratings, popularity, release date, and genres

- Netflix-inspired UI
  - Dark theme with red highlights
  - Responsive design using Tailwind CSS

---

## Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Charts:** Recharts
- **Icons:** lucide-react
- **API:** The Movie Database (TMDB)

---

## Installation & Setup

1. Clone the repository
```bash
git clone https://github.com/your-username/flixstats.git
cd flixstats
```

2. Install dependencies
```bash
npm install
```

3. Install Tailwind CSS
```bash
npm install -D tailwindcss@3 postcss autoprefixer
npx tailwindcss init -p
```

4. Run the development server
```bash
npm run dev
```
